# 1-Forest, Parent & Child Domain, 2-ADSites, 1-Workstation

This Templates deploys a Single Forest with a Parent & Child Domain, 2 Active Directory Sites with a Domain Controller in each, Organizational Units, DNS Records and 1 Domain Joined Windows 10 Workstation in the Child Domain
