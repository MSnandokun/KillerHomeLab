# 1-Forest/Domain, 2-DomainControllers, 2-ADSites, 2-MemberServers, 2-Azure Recovery Services Vaults, 2-Azure Key Vaults

This Templates deploys a Single Forest/Domain, 2 Active Directory Sites with a Domain Controller in each.  Each Site has a Domain-Joined Member Server that will be encrypted by it's Sites Azure Key Vault and protected by it's Azure Recovery Services Vault.

Click the button below to deploy
